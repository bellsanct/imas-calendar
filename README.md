# aonori-imas-calendar

プロデューサースケジュール表から GoogleCalendar へ自動でイベント登録するもの。

---
## 実行方法

1. 事前に GoogleCalendarAPI の設定 json ファイルを用意する
1. 該当 json ファイルを本プロジェクトディレクトリ直下に配置し、各ソースの$aimJsonPath を該当ファイル名にする
1. `php run.php`
